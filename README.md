# 📝 To-Do List Application

![App Screenshot](Screenshot%20(1063).png)

A simple and efficient task management application that allows users to create, update, and manage daily tasks. Designed using ASP.NET Core MVC and Entity Framework, the app focuses on clean structure, usability, and reliable performance.

---

## 🚀 Features
- Add, edit, and delete tasks  
- Mark tasks as completed  
- Organize daily activities efficiently  
- Clean and user-friendly interface  

---

## 🛠 Tech Stack
- C#  
- ASP.NET Core MVC  
- Entity Framework Core  
- SQL Server  

---

## 📌 Key Highlights
- Built using MVC architecture  
- Implements CRUD operations  
- Uses LINQ for database queries  
- Structured and maintainable code  

---

## 🔗 Future Improvements
- User authentication  
- Task categories & priorities  
- Due dates & reminders  

---

## 📂 How to Run
1. Clone the repository  
2. Open in Visual Studio  
3. Configure SQL Server connection  
4. Run the application
